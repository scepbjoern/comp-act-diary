# Technologie-Stack

Dieses Projekt verwendet einen festen Stack. **Keine Alternativ-Frameworks vorschlagen**, ausser explizit angefragt.

---

## Core Framework

| Technologie | Version | Zweck |
|-------------|---------|-------|
| **Next.js** | 14+ | Full-Stack Framework, App Router |
| **React** | 18+ | UI Library |
| **TypeScript** | 5+ | Typisierung |

---

## Styling & UI

| Technologie | Zweck |
|-------------|-------|
| **Tailwind CSS** | Utility-first CSS |
| **daisyUI** | Komponenten-Library (Buttons, Cards, etc.) |
| **@tabler/icons-react** | Icon-Library |
